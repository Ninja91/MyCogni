DESIGN RESPONSE

MyCogni preserves the denominator: not checked, ambiguous, blocked, and inconclusive remain visible. A broker's “completed” response is never silently promoted to verified removal.

04 System architecture

Extensions do less.

AUTHENTICATED USER CONTROL

The user establishes intent and exceptions.

The CLI or local web interface authenticates the actor, records separate scan consent, previews exact disclosures, and requires a dedicated step-up per-capability ceremony before automation can leave global pause.

CAN

Authorize bounded actions, pause, revoke, export, delete.

CANNOT

06 Delivery path

Evidence-gated, not calendar-gated

A 40-week-plus route
to a narrow local v1.

The week ranges show order and size. Passing safety, legal, usability, and evidence gates—not elapsed time—unlocks the next release.

WEEKS 0–4

Build the foundation that can say “no.”

Land locked project boundaries, a deterministic simulator, hermetic network-deny CI, stable threat IDs, and executable security spikes.

EXIT GATES
  • P0 ADRs accepted
  • Synthetic-only test path
  • Threat-to-requirement traceability

Cloud-small, OpenClaw, and optional local-assist evaluation come after v1. The week ranges are a planning envelope, not a promise; the completion matrix and evidence gates are authoritative.

CURRENT

No accepted deployable image yet.

Executable foundations and the synthetic simulator are integrated. GOV-001 and NET-001 have code-level acceptance, while authenticated attestations remain open. Auth and runner are active, unaccepted spikes; key, durability, backup, egress, and browser work plus Docker multi-architecture evidence remain open.

V1 TARGET

Local-lite, single tenant.

A pinned all-in-one container should run sporadically on a laptop or small CPU host. Cloud-small, OpenClaw integration, and optional local inference remain post-v1 evaluation paths—not current capabilities.

07 Adversarial review

Five independent critique tracks

The design was attacked
before code was trusted.

These were AI-assisted role reviews—not claims of human security certification. Each track reviewed the same snapshot independently before the principal-role synthesis.
