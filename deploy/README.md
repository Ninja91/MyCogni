# Deployment artifacts

This directory will contain the core OCI build, separate connector/browser artifact examples, local Compose profile, cloud-small profile, mandatory egress-gateway policy, and hardening guidance during the implementation roadmap. The architecture phase intentionally does not include a pretend Dockerfile for an application that has not been implemented.

See [docs/07-deployment-architecture.md](../docs/07-deployment-architecture.md) for the target and acceptance criteria.
