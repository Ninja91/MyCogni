# Support

MyCogni has no supported release and no service-level agreement. The repository is currently an architecture/specification pack.

Use a public GitHub discussion or issue only for synthetic, non-sensitive product, documentation, or implementation questions. Never post names, addresses, emails, phone numbers, dates of birth, broker-profile URLs, screenshots, authorization records, inbox content, credentials, cookies, or real evidence.

- Security vulnerability or sensitive exploit: use a private GitHub security advisory as described in [SECURITY.md](SECURITY.md).
- Suspected connector drift: file the connector issue form with public procedure URLs and synthetic reproduction only.
- Urgent personal-safety situation: do not rely on this volunteer project for emergency response. Contact appropriate local services or a qualified safety/privacy professional.
- Legal question: project documentation is technical product guidance, not legal advice or representation.

Maintainers may close or remove public material that creates a privacy or safety risk.
